# ArenaSync

ArenaSync is a web-based platform designed for gaming enthusiasts to discover, explore, and participate in various gaming events and activities. The application provides an intuitive interface for browsing different game categories, managing user accounts, and accessing support resources.

## Features

- **Home Page**: Welcome interface with navigation to different sections
- **Events**: Browse and explore gaming events across multiple categories including adventure, horror, RPG, simulation, sports, and strategy games
- **Login System**: User authentication for personalized experiences
- **Support**: Help and support resources for users
- **Responsive Design**: Optimized for various screen sizes with dedicated CSS styling

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and responsive design
- **JavaScript**: Interactive functionality
- **XAMPP**: Local development server environment

## Installation and Setup

1. **Prerequisites**:
   - XAMPP installed on your system
   - Web browser (Chrome, Firefox, Safari, etc.)

2. **Setup**:
   - Clone or download the project files
   - Place the `ArenaSync` folder in your XAMPP `htdocs` directory (e.g., `C:\xampp\htdocs\ArenaSync`)
   - Start XAMPP and ensure Apache is running

3. **Access the Application**:
   - Open your web browser
   - Navigate to `http://localhost/ArenaSync/`
   - The main page (`index.html`) will load automatically

## Usage

- **Navigation**: Use the main navigation to switch between Home, Events, Login, and Support sections
- **Events**: Browse different gaming categories and view available events
- **Login**: Access user-specific features (authentication system)
- **Support**: Find help and resources for using the platform
